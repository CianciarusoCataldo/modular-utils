## Getting started

### Installation

If you want to use this library inside your project, just install it:

```sh
npm i modular-utils
```

<br>

---

## Tests

Unit tests for every lib function are located inside `tests` folder. The test script is executed with pre-defined test command:

```sh
npm run test
```
